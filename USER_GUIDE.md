# Sports Central - User Guide

Welcome to Sports Central! This guide will help you get started with the application.

## Getting Started

### Sign Up / Sign In

1. **Sign in with Google** (Recommended)
   - Click the "Sign in with Google" button
   - Select your Google account
   - Grant permissions
   - You're in!

2. **Sign up with Email**
   - Click "Don't have an account? Sign Up"
   - Enter your full name
   - Enter your email address
   - Create a password (minimum 6 characters)
   - Click "Sign Up"

3. **Sign in with Email**
   - Enter your email address
   - Enter your password
   - Click "Sign In"

## Features

### For All Users

#### Viewing Events
- **Tabs**: Use the tabs at the top to filter events:
  - **Upcoming**: Events that haven't started yet
  - **Live**: Events currently in progress
  - **Completed**: Finished events with final scores
  - **All**: View all events

#### Event Information
Each event card displays:
- Sport type (Football, Basketball, Cricket, etc.)
- Event name
- Date and time
- Venue location
- Team names
- Current scores
- Event status (Scheduled, Live, Completed)
- Winner (for completed events)

#### Real-time Updates
- Scores update automatically when admins make changes
- No need to refresh the page
- Live events are highlighted with a pulsing indicator

### For Admin Users Only

#### Creating Events
1. Click the "New Event" button in the top right
2. Fill in the event details:
   - Event name
   - Sport type
   - Date and time
   - Venue
   - Team 1 name
   - Team 2 name
3. Click "Create Event"

#### Starting a Live Event
1. Find the scheduled event
2. Click "Start Live"
3. The event status changes to "Live"
4. You can now update scores

#### Updating Scores
1. Click "Update Score" on a live event
2. Enter scores for each team
3. For cricket matches, also enter:
   - Wickets (out of 10)
   - Overs (e.g., 15.3 for 15 overs and 3 balls)
4. Click "Update" to save changes
5. Click "Complete" when the event is finished

#### Editing Events
1. Find the event you want to edit
2. Click "Edit" (available on completed events)
3. Make your changes
4. Click "Update" or "Complete"

#### Deleting Events
1. Find the event you want to delete
2. Click the "X" button in the top right of the event card
3. Confirm the deletion

## Troubleshooting

### Can't Sign In
- Check your email and password
- Make sure you're using the correct sign-in method (Google vs Email)
- Try resetting your password (contact admin)

### Not Seeing Updates
- Check your internet connection
- Refresh the page
- Sign out and sign back in

### Need Admin Access
- Contact your system administrator
- Provide your email address used for sign-up

### Events Not Loading
- Check your internet connection
- Make sure you're signed in
- Try refreshing the page
